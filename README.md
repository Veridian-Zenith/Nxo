# Nxo Launcher

A Niagara-inspired launcher for Android 14-17+, designed with Material Expressive principles, transparency, glass effects, and AI-driven app sorting.

> [!WARNING]
> **This project is currently a Work In Progress (WIP). It is not ready for beta release and may be unstable.**
